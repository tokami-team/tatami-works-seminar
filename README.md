# Tatami Works セミナー資料

Tatami Works主催のClaude Code／AI活用セミナーで使用するスライド・配布資料を管理するリポジトリです。

## 公開URL

- **セミナースライド**: https://tokami-team.github.io/tatami-works-seminar/
- **プラン詳細ガイド（参加者向け）**: https://tokami-team.github.io/tatami-works-seminar/plan-details.html

## 収録資料

| ファイル | 用途 |
|---------|------|
| `index.html` | セミナー本編のプレゼンスライド |
| `plan-details.html` | セミナー参加者向けのプラン詳細ガイド（スポット相談／目的別コース／月額プラン） |
| `assets/` | スライド・ガイドで使用する画像一式 |
